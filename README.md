# チュータスズキさんの高低アクセント | Pitch Accent by University of Tokyo Graduate School of Engineering

> This is an attempt to redesign one of the most popular website for learning Japanese pitch accent.
> I use the same API call to the same server. **This is for educational purpose and I have no intention of making profit or turning this into any business.**

- A minimal version of 韻律読み上げチュータスズキクン (without the audio)
